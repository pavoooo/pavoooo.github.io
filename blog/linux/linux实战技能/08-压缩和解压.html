压缩和解压

# 打包

linux系统中文件的打包操作主要是通过tar命令完成

tar命令的操作都是通过选项进行的，tar命令的选项是不需要指定-符号的

tar 选项 打包后的文件名称 需要打包的文件
1
  • c：对文件进行打包
  • x：对打包后的文件进行解包
  • f：指定操作类型为文件
  • z：对打包后的文件进行gzip压缩
  • j：对打包后的文件进行bzip2压缩
  • -C：指定文件解包之后的位置
# 将tmp文件打包成tmp.tar
tar cf tmp.tar tmp
1
2
# 将tmp文件打包并使用`gzip`压缩
tar cfz tmp.tar.gz tmp
1
2
# 将tmp文件打包并使用`bzip2`压缩
tar cfz tmp.tar.bz2 tmp
1
2
# 对tmp.tar进行解包，并将解包后的文件放在/root目录下
tar xf tmp.tar -C /root
1
2

当然，你也可以对压缩的文件进行解压后再进行解包

tar xfz tmp.tar.gz -C /root
1

.tar.bz2可以缩写为.tbz2。.tar.gz可以缩写为.tgz

# 压缩和解压缩

压缩我们通常使用的命令时gzip和bzip2。gzip压缩的更快，但是压缩效率相对于bzip2不高。bzip2压缩的要慢一些，但是压缩效率相对于gzip很高。

# gzip

# 压缩file文件
gzip file # file文件压缩后就成了file.gz
1
2

gzip压缩文件的时候可以指定压缩效率。压缩效率是一个介于1－9的数值，预设值为"6"，指定愈大的数值，压缩效率就会愈高。

gzip file -9
1
# 解压file.gz文件
gzip -d file.gz
1
2
# 压缩file文件，并指定压缩效率为9
gzip file -9
1
2
gzip file --best # --best类似于-9
gzip file --fast # --fast类似于-1
1
2

# bzip2

# 使用bzip2算法压缩file文件
bzip2 file
1
2
# 解压缩bzip2压缩的文件
bzip2 -d file.bz2
1
2

gzip和bzip2只能对文件进行压缩。如果需要对目录进行压缩需要先使用tar命令对目录进行打包。
