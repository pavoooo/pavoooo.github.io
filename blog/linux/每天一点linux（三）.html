每天一点linux（三）

今天要学习的是和几个文档有关的命令，在我们查看文件内容或者实时查看日志的时候会很有帮助。

# cat

cat命令主要是用来查看文件内容的。

这个命令还有几个比较常用的选项：

  • -n：查看文件内容的时候把行号也显示在屏幕上

  • -A：显示所有的内容，包含一些特殊字符。

# tac

tac是一个比较有意思的命令。从字面上来看它是cat命令的逆序书写。其作用也是和cat命令相反，它用于倒序显示文件的内容。-n和-A选项不适用它。

# more

more命令主要用在所查看的文件内容过多的场景下使用。如果文件内容过多，它只会显示一屏的内容。我们可以按空格键继续查看下一屏，或者按q键退出查看。

more index.html
1

# less

less命令和more命令的作用比较类似。但less命令对输出的文本提供了更多的交互。

  • 空格键：实现翻页
  • j键：向下移动
  • k键：向上移动
  • /：向下查找字符，按n键显示下一个
  • ?：向上查找字符，按n键显示上一个
/root # 查找 
?root

# head

head这个命令主要是用于显示文件的前10行。使用-n可以指定需要显示的行数。

显示文件的前5行

# tail

tail这个命令默认会显示文件的后10行。同样可以使用-n指定需要显示的行数.

显示文件的后3行

tail的-f选项也是十分常用的。它会动态监测文件的内容，当文件内容有变化的时候实时输出到屏幕上。这对我们实时检测日志很有帮助。
