每天一点linux（八）

昨天学习了一些关于shell的基础知识，今天就需要了解一下shell编程。shell编程也被称为shell脚本，其运行环境就是linux的shell。

那什么是shell脚本呢？shell脚本就是逻辑+一些命令的集合。

那我们怎么创建和执行shell脚本呢？首先，shell脚本就是一个普通的文本文件，通常以.sh结尾。在shell脚本的首行，通常需要添加#!/bin/bash，表明这个脚本所使用的是bash语法。下面就是一个普通的shell脚本。

#!/bin/bash

# 这是我们第一个shell脚本
echo 第一个shell脚本文件
1
2
3
4

shell脚本中，#表示注释

编写好shell脚本后，我们就需要执行她。执行的方式有两种：

第一种：shell名称 脚本路径

bash first.sh
1

使用这种方式来执行shell脚本，可以通过-x选项来查看脚本执行的过程

bash -x first.sh
1

第二种：基本路径，这种情况下需要赋予这个脚本可执行权限

chmod +x ./first.sh
./first.sh
1
2

# 变量

shell脚本中也有变量的概念。shell脚本变量的定义格式是

变量名=变量值
1

WARNING

定义变量的时候，变量名和变量值之间等号两边不能有空格

使用变量直接$变量名即可。

name=saisai
echo $name
echo '$name' # ''的作用和先前介绍的是一样的
echo "$name" # ""的作用和先前介绍的也是一样的
1
2
3
4

我们也可以把一个命令的输出结果赋值给一个变量

directory=`pwd` # ``会把其中的字符串当作命令来执行
echo $directory
1
2

# 数学运算

我们可以在shell脚本中进行简单的数学运算。进行数学运算需要遵循如下的格式：

$[进行数学运算的式子]
1
a=1
b=2
c=3

echo $[$a+$b+$c]
echo $[1+$a+$b*$c]
1

# 读取用户输入

我们可以在shell中通过read命令来和用户进行交互。通过-p选项输出提示语。用户输入的结果会赋给我们所指定的变量。

read -p '提示语' 变量名
1

比如

read -p '请输入第一个数字：' first
read -p '请输入第二个数字' second

echo '计算结果是：'
echo $first+$second
1
2
3
4
5

# 预设变量

预设变量就是在shell脚本中预先定义好的具有特殊含义的变量。比如，我们的shell脚本如下

echo $0 $1 $2
1

其中的$0，$1，$2...等就是脚本中预设的变量。这些变量都是在我们执行脚本的时候传递的。比如

bash prev.sh 1 2 # $0 就是 prev.sh $1 就是1 $2 就是2等等
1

# if

if 判断语句; then
  command
fi
