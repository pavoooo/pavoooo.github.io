pwd和ls命令

linux中一切皆文件，文件的操作都是通过命令进行

# pwd

pwd用于显示当前的操作目录

[root@instance-6w9riila ~]# pwd
/root
1
2

# ls

ls用于显示目录的内容

ls [options] [args...]
1
[root@instance-6w9riila ~]# ls /
bin  boot  cgroup  dev  etc  flag_ok  home  lib  lib64  lost+found  media  mnt  opt  proc  root  run  sbin  selinux  srv  sys  tmp  usr  var

第一个字符具有特定的含义

  • -：表示该文件是一个普通文件
  • d：表示该文件是一个目录文件
