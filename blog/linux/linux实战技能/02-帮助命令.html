帮助命令

# man

man是manual的缩写

# 基本使用
man 命令名称
1
2
man ls
1

有些命令是分为好几个篇章，通过man也可以查看某个命令的具体篇章

man 7 man
1

上面的命令就是查看man命令的第7篇章。

也可以通过-a选项让man命令找出所有和指定关键字相关的内容

man -a passwd
1

# help

shell自带的名称成为内部命令，其他的是外部命令。

内部命令我们可以使用help查看其帮助

help cd
1

外部命令的help帮助使用如下

ls --help
1

内部命令和外部命令我们可以使用type命令来判断

[root@instance-6w9riila ~]# type pwd
pwd is a shell builtin
[root@instance-6w9riila ~]# type ls
ls is aliased to `ls --color=auto'
1
2
3
4

info 命令名称
